# stockzoa
A web-based stock manager build with Angular js , Materialize CSS and PHP.

Steps for deployment:

1. Fork and Clone the repo.
2. Place inside the xampp/htdocs folder
3. Import the sql file in phpmyadmin
4. Run the server
